## 🚀 デプロイ手順

1. **最終テスト**
   ```bash
   npm run test:expo
   ```

2. **環境変数確認**
   ```bash
   npm run check-env
   ```

3. **EASビルド（プレビュー）**
   ```bash
   npm run eas-build-preview
   ```

4. **内部テスト配信**
   - TestFlight（iOS）
   - Internal Testing（Android）

5. **本番ビルド**
   ```bash
   npm run eas-build-production
   ```

6. **ストア申請**
   - App Store Connect
   - Google Play Console
